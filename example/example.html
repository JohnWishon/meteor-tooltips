Tooltips for Meteor

Demonstrating Meteor tooltips, by Lookback.

  • GitHub
  • Atmosphere
  • Source for this demo

Simply embed the tooltips template in your master template/layout and attach tooltips with adding the data-tooltip attribute with the tooltip text.

Default tooltip.

<button data-tooltip="I'm a tooltip!">A tooltip trigger</button>

Tooltip to the left (west). Data attribute values:

  • n - Top (default).
  • s - Bottom.
  • e - Right.
  • w - Left.
<button data-tooltip="I'm a tooltip!" data-tooltip-direction="w">A tooltip trigger</button>



Offset tooltip by specifying the data-tooltip-top and data-tooltip-left attributes. Both attributes take plain numbers (negative and positive), counted in pixels.

The end result of this also depends on data-tooltip-direction. 

<button data-tooltip="I'm a tooltip!" data-tooltip-top="50">A tooltip trigger</button>

<button data-tooltip="I'm a tooltip!" data-tooltip-top="-50">A tooltip trigger</button>

<button data-tooltip="I'm a tooltip!" data-tooltip-direction="w" data-tooltip-left="50">A tooltip trigger</button>

<button data-tooltip="I'm a tooltip!" data-tooltip-direction="e" data-tooltip-left="50">A tooltip trigger</button>
Nested element

Tooltips should work as expected when attaching them to a parent element. 

<span data-tooltip="Top-level text"><strong>Nested element</strong></span>
